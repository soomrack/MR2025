# 1. Описание

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  Chat MAX 10.0 — многопользовательский чат-сервер на языке C работающий     │
│  работающий на Raspberry Pi 5 / Raspbian.                                   │
│                                                                             │
│  Сервер принимает TCP-подключения на нужном порту и одновременно управляет  │
│  гученичным роботом через UART-соединение с Arduino Mega 2560.              │
│  Пользователи могут отдавать команды движения прямо из терминала чата.      │
└─────────────────────────────────────────────────────────────────────────────┘
```
